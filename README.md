# OuterProductBenchmarking


This is a repository for a one off lecture on optimising GPU Kernels.

This implements various forms of the outer product operation to teach
high-level operations. It is not intended to provide the best approach,
or a high performance kernel but to be used as a teaching tool.


## Build instructions

There is a simple MakeFile to build the benchmarks.

Install ROCm: https://rocm.docs.amd.com/projects/install-on-linux/en/latest/
git clone the directory and run `make`
